# Boist.Tests

A variety of sample programs that you can run. Simply change this line in Program.cs to look at different tests:

```cs
var test = new ShapeTest(resolvedAPI); // <-- You can change this to the other tests available
```

You can look their source code to learn more about Boist, however they aren't too documented. `ShapeTest.cs` is the most documented.
